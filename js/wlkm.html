SEARCH BOX

- The search box displays suggestion after user types at-least 2 alphabets.
- The search box displays suggestion in in 3 levels the following order(top-down):
State (at most 10)
District, State (at most 10)
Block, District, State (at most 10)

RANGE SLIDER(setting the milestone)

- The range(in percentage) of slider determines :
At state level(from search box), displaying those districts that have at-least one habitation reaching the milestone.
At district/block level(from search box), displaying those blocks that have at-least one habitation reaching the milestone.
At habitation level data of each block, displays list of habitation reaching the milestone.

MAP ALERT

- The circular markers(red[>=100],yellow[>=10],green[<10]) display the grouping of markers(marker clusters).
- Number on the circle displays how many markers are clustered under it and the blue polygon displays the way markers are spread over map.
- On selecting state from search suggestion, districts of that state are plotted on map.
- On selecting district from search suggestion, blocks of that state are plotted on map.
- On selecting district from search suggestion, blocks of that state are plotted on map.
